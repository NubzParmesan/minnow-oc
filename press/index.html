What is minnow_oc?

minnow_oc is a small static site and starter kit for Tuna: a constraint-first approach to memory and identity for amnesiac AI agents. It collects a canonical reference (canon/doctrine/blessings) plus practical templates you can copy into a repo and run with. The goal is simple: make onboarding fast, links stable, and agent behavior easier to reproduce.
